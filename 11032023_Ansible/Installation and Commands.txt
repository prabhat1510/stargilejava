Install Node.js and set up the express server
 curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
 . ~/.nvm/nvm.sh
 nvm install --lts
mkdir server && cd server
npm install express
If you get error after npm install express then uninstall the nvm using below command
 nvm uninstall 18.15.0

Then install stable version of node 
nvm install 16.0.0

npm install express
